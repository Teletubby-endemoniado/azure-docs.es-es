Cualquier usuario que tenga acceso al área de trabajo puede etiquetar los datos del proyecto.  También puede personalizar los permisos de los etiquetadores para que puedan acceder al etiquetado, pero no a otras partes del área de trabajo o del proyecto de etiquetado.  Para obtener más información, consulte [Administración del acceso a un área de trabajo de Azure Machine Learning](../articles/machine-learning/how-to-assign-roles.md) y aprenda a crear el [rol personalizado de etiquetador](../articles/machine-learning/how-to-assign-roles.md#labeler).

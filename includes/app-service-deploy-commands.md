* [Administración de recursos de Azure con Azure PowerShell](../articles/azure-resource-manager/management/manage-resources-powershell.md)
* [Administración de recursos de Azure mediante la CLI de Azure](../articles/azure-resource-manager/management/manage-resources-cli.md).

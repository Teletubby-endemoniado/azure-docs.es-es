# <a name="what-is-the-azure-devtest-offer"></a>¿Qué es la oferta de desarrollo/pruebas de Azure?

En este artículo se explica la oferta de desarrollo y pruebas de Azure que se incluye con las ventajas de la suscripción de Visual Studio. Con esta oferta, recibe créditos mensuales de Azure para explorar y probar varios servicios de Azure sin costo alguno.  La cantidad de crédito varía según el nivel de suscripción.  Los usuarios también pueden beneficiarse de las tarifas de desarrollo y pruebas de Azure con descuento en varios servicios de Azure, lo que permite que estas tareas salgan rentables.

## <a name="who-can-benefit"></a>Quién puede beneficiarse? 

Con la oferta de desarrollo y pruebas de Azure, estos suscriptores de Visual Studio pueden usar Azure:
+ [Los usuarios pueden usar Azure como espacio aislado personal para desarrollo y pruebas](quickstart-individual-credit.md). 
+ [Las empresas pueden usar Azure para el desarrollo en equipo y la colaboración](quickstart-create-enterprise-devtest-subscriptions.md).

# <a name="azure-policy-built-in-packages-for-guest-configuration"></a>Paquetes integrados de Azure Policy para la configuración de invitados

Esta página es un índice de paquetes integrados de Azure Policy para la característica de configuración de invitado.

## <a name="how-to-use-guest-configuration-package-details"></a>Cómo usar los detalles del paquete de configuración de invitado

Cada fila representa un paquete utilizado por una definición de directiva integrada.

- **Definición**: conduce a la definición de directiva en Azure Portal.
- **Configuración:** conduce al archivo `.mof` del [repositorio de GitHub de Azure Policy](https://github.com/Azure/azure-policy) que contiene la configuración que se usa para auditar o corregir máquinas.
- **Módulos necesarios:** conduce a los módulos de [Desired State Configuration (DSC) de PowerShell](/powershell/scripting/dsc/overview/overview) que usa cada configuración. Los módulos de recursos contienen la lógica de script utilizada para evaluar cada parámetro de la configuración.
  
Para comprender qué parámetros se comprueban en Windows o Linux, y cómo, busque el nombre de la definición de directiva en la columna izquierda y vaya al recurso de DSC de la columna derecha para revisar los scripts de PowerShell.

La tabla no incluye detalles de los paquetes usados para evaluar las configuraciones de base de referencia. Las bases de referencia se escriben en C++, en lugar de en Desired State Configuration de PowerShell.

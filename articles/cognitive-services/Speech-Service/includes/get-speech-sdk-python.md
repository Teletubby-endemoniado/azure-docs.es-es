```Bash
pip install azure-cognitiveservices-speech
```

> [!TIP]
> Si está en macOS, es posible que tenga que ejecutar el siguiente comando para que el comando pip anterior funcione:
> ```Bash
> python3 -m pip install --upgrade pip
> ```
